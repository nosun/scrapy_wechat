## 说明
本项目为学习使用，欢迎一起交流研究。

## 项目方案
通过预先设定一批公众号的 id，通过搜狗的搜索入口，进入微信的 “10篇最新文章” 列表页，进行爬取。

## 项目特点
目前没什么特点，如果不遇到验证码的情况，可以正常爬取，验证码的问题还有待解决。
